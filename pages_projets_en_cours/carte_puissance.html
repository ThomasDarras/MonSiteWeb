    • CARTE PUISSANCE SERVO MOTEUR

        
    Ce projet est dans la continuité du bras robot 2.0.

    
    Objetifs
    
    Créer une carte permettant l'alimentation de 6 servo moteurs ainsi que de pouvoir générer des signaux PWM via code dont les instuctions sont donnés par bluetooth.

    Le code sera en C++ pour avoir une execution rapide et pouvoir gérer le bluetooth.

    L'alimentation de la carte sera faite en continue avec 230V.

    
    Etat du projets
    
    L'analyse des besoins est en cours.

    Le choix du micro procresseur n'est pas encore fait.
